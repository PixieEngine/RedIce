Fadeable(I, self)
The Fadeable module provides a method to fade a sprite to transparent. You may also provide a callback function that is executed when the sprite has finished fading out.
Parameters:
I Object
Instance variables
self Core
Reference to including object
fadeOut(duration, callback)
A convenient way to set the fade instance variables on a sprite. You can modify the instance variables by hand but the suggested way to do it is through this method. 
    player = GameObject()

    player.include(Fadeable)

    fadedOut = false

    # this will fade the player object out over the next 30 frames. 
    # once the player is faded out the fadedOut variable will be set to true.
    player.fadeOut 30, (player) ->
      fadedOut = true

    30.times ->
      player.update()

    fadedOut
    # => true
Parameters:
duration Number Optional, Default: 30
How long the effect lasts
callback Function Optional, Default: null
The function to execute when the sprite has finished fading.
