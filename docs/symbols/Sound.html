Sound()
A simple interface for playing sounds in games.
static Sound.play(id, maxChannels)
Play a sound from your sounds directory with the name of `id`. 
    # plays a sound called explode from your sounds directory
    Sound.play('explode')
Parameters:
id String
id or name of the sound file to play
maxChannels String
max number of sounds able to be played simultaneously
static Sound.playFromUrl(url)
Play a sound from the given url with the name of `id`. 
    # plays the sound at the specified url
    Sound.playFromUrl('http://YourSoundWebsite.com/explode.wav')
Parameters:
url String
location of sound file to play
Returns Sound :
this sound object
static Sound.stop(id)
Stop a sound while it is playing. 
    # stops the sound 'explode' from 
    # playing if it is currently playing 
    Sound.stop('explode')
Parameters:
id String
id or name of sound to stop playing.
