Local()
A wrapper on the Local Storage API
static Local.get(key)
Retrieve an object from local storage. 
  Local.get('name')
  # => 'Matt'

  Local.get('age')
  # => 26

  Local.get('person')
  # => { age: 26, name: 'Matt' }
Parameters:
key String
string that identifies the stored object
Returns Object :
The object that was stored or undefined if no object was stored.
static Local.new(prefix)
Access an instance of Local with a specified prefix.
Parameters:
prefix String
Returns Local :
An interface to local storage with the given prefix applied.
static Local.set(key, value)
Store an object in local storage. 
  # you can store strings
  Local.set('name', 'Matt')

  # and numbers
  Local.set('age', 26)

  # and even objects
  Local.set('person', {name: 'Matt', age: 26})
Parameters:
key String
string used to identify the object you are storing
value Object
value of the object you are storing
Returns Object :
value
