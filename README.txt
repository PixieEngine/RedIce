= Red Ice

The hockey game that bleeds fun.

= To Play

Plug in some gamepads.

Open `run.html` in Google Chrome

== Controls

=== Keyboard

Move - Arrow Keys
Turbo - Z
Shoot - X

=== Gamepad

Move - Joystick/D-Pad
Turbo - A, Y
Shoot - B, X

Charge up shot for maximum power.
